Les applications utilisant des APIs externes représentent une excellente manière d'apprendre le développement frontend moderne.

CID Movie est un projet réalisé avec React et Vite dont l'objectif était de créer une plateforme permettant de découvrir des films grâce aux données fournies par l'API TMDB.

<Callout>Le fonctionnement du projet</Callout>

L'application récupère les informations des films depuis TMDB puis les affiche dans une interface pensée pour faciliter la découverte.

Le projet m'a permis de travailler sur plusieurs aspects :

* récupération de données externes ;

* affichage dynamique ;

* organisation des composants React ;

* gestion des interfaces utilisateur.

<Callout>Les défis rencontrés</Callout>

L'un des principaux défis était de gérer correctement les données provenant de l'API.

Il fallait notamment réfléchir à :

* la structure des données ;

* l'affichage des informations ;

* la gestion des états ;

* l'expérience utilisateur.

<Callout>Ce que j'ai appris</Callout>

Ce projet m'a permis de mieux comprendre comment connecter une application frontend à un service externe et comment transformer des données brutes en une expérience utilisateur claire.
